API: Upload PFX File

To upload your PFX file, use the following API:

POST https://mxserversign.managexindia.in/upload
                                
                                Body: form-data
                                Key: file
                                Type: file
                                Value: Select the PFX file you wish to upload

API: Sign a PDF

To sign a PDF, use the following API:


                                POST https://mxserversign.managexindia.in/sign
                                Body: raw JSON
                                
                                
                                {
                                    "request": {
                                        "command": "managexserversign",  // Mandatory
                                        "timestamp": "",  // Mandatory: Send ISO timestamp
                                        "transaction_id": "",  // Mandatory: Ensure no duplicates
                                        "parameter": {
                                            "webhook_url": ""  // Optional: If you want the response sent to a webhook
                                        },
                                        "pfx": {
                                            "pfx_name": "",  // PFX file name (without the .pfx extension)
                                            "pin": ""  // Password for the PFX file
                                        },
                                        "pdf": {
                                            "coordinates": "",  // Coordinates for signing (optional) if "search_by_text"
                                            "search_by_text": "",  // Target text to search for in PDF (optional) if "coordinates"
                                            "page": "",  // Page number to sign on like 1 or you can user first or last to sign pdf 1st page or last page 
                                            "reason": "",  // Reason for signing
                                            "custom_text": "",  // Custom text for signature box
                                            "location": "",  // Location for the signature
                                            "dateformat": "",  // Date format
                                            "lockpdf": "",  // "yes" to lock the PDF after signing
                                            "invisiblesign": "",  // "yes" for invisible signature
                                            "email": "",  // Email to send signed document
                                            "recipient": "",  // Recipient's name
                                            "title": ""  // Title of the signed document
                                        },
                                        "pdf_data": "",  // Base64 encoded PDF or URL to the PDF
                                        "pdfurl": ""  // Alternatively, provide PDF URL
                                    }
                                }

API for Date Format

To use the dateformat field, here are the allowed formats:


                                {
                                    "request": {
                                        "pdf": {
                                            "dateformat": "dd-MMM-yyyy"  // This is the example date format to indicate where you want to make changes in your request.
                                        }
                                    }
                                }

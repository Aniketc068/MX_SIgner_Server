API: Upload PFX File

To upload your PFX file, use the following API:

POST http://127.0.0.1/upload
                                                                    
                                    Body: form-data
                                    Key: file
                                    Type: file
                                    Value: Select the PFX file you wish to upload
                                    
                                    Key: pin
                                    Type: text
                                    Value: Enter the PIN associated with the PFX file

API: Sign a PDF

To sign a PDF, use the following API:


                                POST http://127.0.0.1/sign/api/v1.0/postjson
                                Body: raw JSON
                                
                                
                                {
                                    "request": {
                                        "command": "managexserversign",  // Mandatory
                                        "timestamp": "",  // Mandatory: Send ISO timestamp
                                        "transaction_id": "",  // Mandatory: Ensure no duplicates
                                        "pfx": {
                                            "SN": "" // Mandatory: Uploded Certificate Serial no.
                                        },
                                        "pdf": {
                                            "coordinates": ""  // Coordinates for signing
                                        },
                                        "pdf_data": ""  // Base64 encoded PDF
                                    }
                                }
